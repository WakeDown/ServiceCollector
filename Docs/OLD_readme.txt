1) Для запуска программы необходимо поместить файл лицензии (license.un1t) в папку Program.
2) Для настройки конфигурации необходимо запустить ярлык "Start config"
3) Для установки программы как сервиса необходимо выполнить команду вида:
	C:\Windows\Microsoft.NET\Framework\v4.0.30319\installutil.exe D:\...\UN1TCounter\Program\UN1TCounter.exe
Выполнять необходимо из под учетной записи администратора
4) Для удаления сервиса необходимо выполнить команду вида:
	C:\Windows\Microsoft.NET\Framework\v4.0.30319\installutil.exe /u D:\...\UN1TCounter\Program\UN1TCounter.exe
Выполнять необходимо из под учетной записи администратора
