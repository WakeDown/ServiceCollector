1) Для запуска программы необходимо поместить файл лицензии (license.un1t) в папку Program.
2) Для настройки конфигурации необходимо запустить UN1TCounterSetup.exe
3) Перед установкой программы как сервиса необходимо разместить программу в системной папке Windows - Program files
4) Для установки программы как Windows службы необходимо запустить ServiceInstaller.exe от имени Администратора
5) Для удаления программы из списка служб Windows необходимо запустить ServiceUnnstaller.exe от имени Администратора
